# ArcMapAddin
Provide ArcMap with a plug-in to assist the conversion of CAD to shp.

适用于ArcMap 10.1，其他版本是否支持未做尝试。

该插件主要包括以下功能：<br>
========
·将shp文件导入个人地理数据库。<br>
·对shp文件进行拓扑检查，并输出错误报告，同时支持在ArcMap中进行错误定位。<br>
·标注(Label)转注记(Annotation)。<br>
·渲染图层。<br>
